One Page Site

Ideal para negócios de pequeno ou médio porte. O site One Page é um site onde você inclui todas as informações que o visitante precisa de forma objetiva e criativa em uma única página.

Wireframe

Wireframe é uma prepresentação visual da estrutura básica de uma pagina. Ele deve ser feito de maneira mais simples possível, mostrando apenas o essencial, como uma espécie de rascunho,sem cores e imagem. Clique no botão ou escaneie o QR Code para obter o wireframe desta página.
